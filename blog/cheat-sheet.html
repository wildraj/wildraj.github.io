HASI Cheatsheet

Numbers

4: integer
4.0: float

Operator Integer Floats
+ 4 + 4 = 8 4.0 + 4.0 = 8.0
- 4 - 2 = 2 4.0 - 2.2 = 1.8
* 4 * 3 = 12 4.0 * 4.4 = 17.6
/ 4 / 4 = 1 4.0 / 4.0 = 1.0
% 4 % 4 = 0 Cannot be used

Strings

Strings are "strings" of characters. With the "" the program can use escape characters and #{variable} within it while the '' can't.
Using the operator '+' strings can concatenate. Ex: "Ruby " + "Rules." = "Ruby Rules."

Arrays

Declaration: array = Array.new or array = []

array_example = [0, 1, 2, 3]

Array indexes for array_example are to the same as their values. Remember that the index of an array starts at 0 NOT 1!!!

Indexes are accessed by brackets as follows:
Ex: array_example[2] = 2

Push method: pushes the item at the end of the array
Ex: array_example.push(4) -> array_example=[0,1,2,3,4]

Pop method: displays the last item of the array and deletes it
Ex: array_example.pop() -> array_example=[0,1,2,3]

Hashes

Declaration: hash = Hash.new or hash = {}

hash_example = {
"key1" => "value1"
"key2" => "value2"
"key3" => "value3"
}

Hashes are like dictionaries where the keys point to a specific value. Keys and values are connected by a hash rocket (=>). Keys can only be strings or symbols but values can be any type of data. To access a value requested, type in the array syntax, not with the index, but the key. Ex: hash_example["key1"] = "value1"
